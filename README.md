# BaghChal
Classic game built with python and PyQt5.

## How to run
- Clone: **git clone https://github.com/nightmaregaurav/BaghChal.git**
- cd into project directory: **cd baghchal**
- Open up terminal
- Hit: **python BaghChal.py**
